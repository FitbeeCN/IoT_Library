
This directory contains a copy of the org.json JSON library, as
retrieved from the following GitHub repository on April 30, 2014:

https://github.com/douglascrockford/JSON-java.git

commit: 48d31b7f5c8e43321e4b2143a8a795c366ace6d9

The "org.json" package names have been changed to "org.telehash.json" to
avoid conflicts with the older "org.json" package found on Android
devices.
